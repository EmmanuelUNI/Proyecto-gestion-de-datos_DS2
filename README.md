
# Proyecto Final — Sistema de Gestión en la Nube

Este repositorio contiene el desarrollo del proyecto final, basado en una solución desplegada en la nube con integración de servicios externos.

## Tecnologías principales

* Frontend: React
* Backend: Python y FastAPI
* API inteligente: Google Gemini
* Base de datos: ROBLE (base de datos en la nube de la universidad)
* Plataforma de despliegue: Microsoft Azure

## Descripción general

El proyecto implementa una solución que combina un backend modular, una interfaz web moderna y una capa de inteligencia artificial proporcionada por Gemini.
La base de datos se conectó directamente a ROBLE, la plataforma de base de datos institucional.
El despliegue final se realizó en Microsoft Azure.

## Créditos

Proyecto desarrollado como entrega final para la materia diseño de software II de la Universidad del Norte.
